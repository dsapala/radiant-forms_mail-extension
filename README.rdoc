= Form Mail Extension for Radiant

This extension requires 
  http://github.com/squaretalent/radiant-form-extension

== Usage
  git clone git://github.com/squaretalent/radiant-form_mail-extension vendor/extensions/form_mail

== Form Config

  mail:
    subject: Hey new email!
    recipients:
      - joe.bloggs@email.com
    from: person@email.com
    sender:
    # Optional, read data from a field instead
    subject_field: title                              #<input type="text" name="title" />
    from_field: name
    reply_to_field: name
    sender_field: name
